How to Use Bulk Add

  • File Upload: Upload a text file (.txt) with one chunk per line
  • Text Area: Paste or type text directly, with one chunk per line
  • Empty lines will be ignored
  • Each line will become a separate chunk in the database
  • Very long lines may be truncated
Example format:
This is the first chunk of text to be recorded.
Here is another sentence for recording.
And this is a third chunk.
